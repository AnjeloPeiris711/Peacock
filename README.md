### Running Locally 🚀
Clone the repos's main branch.
```sh
$ git clone -b main
```
Change directory.
```sh
$ cd 
```
Install dependencies, make sure you have `nodejs` and `npm` installed.
```sh
$ npm install
```
Run the app.
```sh
$ npm start
```
